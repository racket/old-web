Enhancements to file dialogs

Proposed changes:

  • Keypress scrolls to first filename past the current position beginning with that key
  • Up and down arrow keys should do the right thing
  • Eliminate GO button, use <RETURN> key instead
  • Add UPDIR button, a la Win95
  • PageUp and PageDown should scroll by a page at a time.
  • The edit field at the bottom of the dialog should reflect the current listbox selection

    • All of the above have been implemented, and committed to the archive. The dialogs need testing before the 51 release.

    Robby had proposed that <SPC> and <DEL> work as PageDown and PageUp, respectively. I'm not sure such duplication is necessary. I, for one, was not familiar with those key bindings.
